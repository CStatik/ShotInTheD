# Private Key Search System
This project is designed to search for a private key within a defined range using AWS GPUs and precomputed datasets.

## Features
- **Uses Pollard Rho Kangaroo algorithm** for key discovery.
- **Runs on AWS Spot Instances** to minimize costs.
- **Utilizes real-time monitoring via a dashboard.**
- **Supports precomputed data optimization.**
